Class Funcionario
       atributos
	int idFunc
	String nome
	String sobrenome
	date dataNasc
	String email
	String senha
	String func   
	char genero
       métodos
	pedidoPago(double pagamento, int[] idPedido)
	     (exclusivo para entregadores)(é um array para casoso como: um cliente pede 2 pizzas(não vou usar array no sql, ent vai ser assim)) recebe a quantia recebida e o id do pedido, remove o pedidio da tabela e devolve o troco(se o troco for negativo, ele não retira o pedido, senão, remove [pagamento] do preço original do pedido).
	ProduzirPizza(idPizza)
	     (exclusivo para cozinheiros(as))recebe o idPizza e, se possível, cria a pizza e remove os produtos do estoque.Caso contrário, não altera nada.
	AddFuncionario(String nome, String sobrenome, date dataNasc, String email, String senha, String func, char genero)
	     (exclusivo para o gerente)Adiciona um funcionário.
